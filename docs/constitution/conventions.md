## 잠정 규약 (제안, 미합의)
- **문서 헤더**: 모든 state/log 문서 머리에 `last-updated / session-id / author-agent`.
- **session-id 형식(제안)**: `YYYY-MM-DD-<짧은꼬리표>` (예: `2026-06-03-cold-start`).
- **spec 파일명(제안)**: `tasks/specs/SPEC-<번호>-<짧은제목>.md` (예: `SPEC-001-scfm-embedding.md`).
- **report 파일명(제안)**: `tasks/reports/REPORT-<같은번호>-<짧은제목>.md`.
- **커밋 메시지(제안)**: `docs: ...` / `spec: ...` / `report: ...` 접두.
